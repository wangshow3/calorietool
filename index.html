热量计算器

（算法来自卫健委《成人肥胖食养指南2024》）
减脂唯一核心：热量缺口公式（重中之重）

热量差值 = 每日总消耗(TDE) − 当天总热量摄入

✅ 差值＞0 = 热量缺口 → 只有这种情况才能减脂

❌ 差值＜0 = 热量盈余 → 持续长胖

➡ 差值≈0 = 热量平衡 → 体重维持不变

补充：健康每日缺口300~700大卡，每月减 2~4kg。减重依靠温和热量缺口搭配均衡合理膳食实现，严禁极端节食，长期极低热量摄入会损伤代谢、极易反弹。

📌 科学减脂膳食建议（依据《中国居民膳食指南》）

核心原则：严禁极端节食，减脂依靠温和热量缺口+均衡膳食，保护基础代谢与身体健康。

  • 拒绝节食减重：女性每日摄入不要低于1200大卡、男性不低于1500大卡；长期极低热量饮食会降低基础代谢、脱发、月经紊乱、乏力反弹，反而更容易长胖。
  • 温和控制热量缺口：每日300~700大卡缺口为健康区间，不要追求快速减重，每月减重2~4kg是人体可适应的安全速度。
  • 三餐均衡搭配，不跳过任何一餐
    • 主食：替换一半精制米面为杂粮、薯类，每餐一拳分量，维持血糖稳定；
    • 蛋白质：每餐手掌大小瘦肉/鱼虾/鸡蛋/豆制品，充足蛋白保护肌肉、延长饱腹感；
    • 蔬菜：每日不少于500g，绿叶蔬菜占一半以上，多吃低卡高纤维食材。
  • 控制添加糖与油脂：少喝奶茶、碳酸饮料、糕点油炸食品；日常烹饪少油少盐，减少隐形热量摄入。
  • 上班族适配运动方案：优先利用通勤步行、办公室短时站立走动；每周搭配3次专项运动（快走、慢跑、球类、家务均可），动静结合提升每日总消耗。
  • 饮水与作息辅助每日1500~1700ml白开水，少喝含糖饮品；保证7~8小时睡眠，熬夜会升高食欲激素，更容易暴饮暴食。
  • 长期维持思路减脂结束后不要立刻恢复暴饮暴食，缓慢缩小热量缺口直至收支平衡，养成可持续的饮食活动习惯，避免体重反弹。

完整计算逻辑公式

1. 基础代谢率 BMR（静息全天基础消耗）

女性：BMR = 655 + 9.5×体重(kg) + 1.8×身高(cm) − 4.7×年龄(岁)

男性：BMR = 66 + 13.7×体重(kg) + 5.0×身高(cm) − 6.8×年龄(岁)

2. 单项活动热量消耗

单项活动能耗 = MET代谢当量 × 体重(kg) × 活动时长(h)

全天活动总消耗 = 所有活动单项消耗累加总和

3. 每日总消耗 TDE（全天全部热量支出）

TDE = BMR×静态休息占比 + 全天活动总消耗 + 食物热效应（总消耗×10%）

重要说明：静态休息占比无需手动填写，系统根据你填写的主动活动自动计算
