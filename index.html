一、官方全套热量计算公式（《成人肥胖食养指南2024》）

1. 基础代谢率 BMR（静息全天基础消耗）

女性：BMR = 655 + 9.5×体重(kg) + 1.8×身高(cm) − 4.7×年龄(岁)

男性：BMR = 66 + 13.7×体重(kg) + 5.0×身高(cm) − 6.8×年龄(岁)

2. 单项活动热量消耗

单项运动能耗 = MET代谢 × 体重(kg) × 运动时长(h)

全天运动总消耗 = 所有运动单项消耗累加总和

3. 每日总消耗 TDE（全天全部热量支出）

TDE = BMR×非活动占比 + 全天运动总消耗 + 食物热效应（总消耗×10%）

4. 减脂唯一核心：热量缺口公式（重中之重）

热量差值 = 每日总消耗(TDE) − 全天饮食总摄入

✅ 差值＞0 = 热量缺口 → 只有这种情况才能减脂

❌ 差值＜0 = 热量盈余 → 持续长胖

➡ 差值≈0 = 热量平衡 → 体重维持不变

补充：减掉1kg纯脂肪，累计需要7700大卡总热量缺口；健康每日缺口300~700大卡，每月减2~4kg
